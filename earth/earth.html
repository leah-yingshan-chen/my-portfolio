Project Overview


This is a side project I worked on in my spare time — a set of emojis inspired by my capstone thesis project.

In my capstone thesis, I designed a character called Trippi--an AI bot shaped like the Earth. Later, I made some adjustments to make it more suitable for WeChat emojis, which is how Beggy was created. I chose the name Beggy as a playful reference to the Chinese internet slang 我球球了, which is often used in a lighthearted or humorous way to express exaggerated pleading or desperation. Besides, the word earth in Chinese is 地球, which shares the same character as the slang term😝
