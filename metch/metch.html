Key Feedback and Observation

  • Lack of option given to set up the account
  • Tedious sign up process - have to type every preferences
  • Unsure how to use the application and the flow of the application
  • No option to skip certain functions (e.g. Ice breaking games)

improvements.

Visibility of System Status



The lack of system status and tedious filling in of detail may make users feel unsure about the application.

User control and freedom



Matching feature lacks clear options for users to exit or skip unwanted states.

Help and documentation



The app does not have an onboarding page that walks through user journey to provide detailed documentation for the users.

Error prevention



Users are supposed to manually key in their interests and preferences. This is subjected to both slips and mistakes by the users.

Aesthetic and minimalist design



The multiple game options distract the user from the application's main feature: making friends through social interaction.
